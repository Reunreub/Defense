REACTION & MEMORY GAME
=====================

This is a web-based game with three different modes:

1. SINGLE PLAYER REACTION TEST
   - Wait for the countdown
   - When the box lights up, click it as fast as you can
   - Your reaction time will be displayed
   - Best time is saved locally on your PC

2. MULTIPLAYER OFFLINE
   - Two players compete
   - Wait for countdown, then click when your box lights up
   - First to click wins the round
   - Score is tracked for both players

3. MEMORY CHALLENGE
   - Watch as boxes light up in a pattern
   - Repeat the pattern by clicking the boxes in the same order
   - Each level increases the pattern length
   - You get 2 replays per level
   - At higher levels, more boxes are available but only up to 10 light up
   - Best level is saved locally on your PC

HOW TO RUN THE GAME:
1. Save all four files (index.html, style.css, script.js, README.txt) in the same folder
2. Open index.html in your web browser
3. No internet connection required - all files are local

TECHNOLOGY:
- HTML5, CSS3, and vanilla JavaScript
- No external dependencies required
- Uses localStorage to save high scores

DESIGN FEATURES:
- Responsive design that works on desktop and mobile
- Smooth animations and visual feedback
- Intuitive user interface with clear instructions
- Attractive color scheme with visual indicators
